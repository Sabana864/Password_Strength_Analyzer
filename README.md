Password Strength Analyzer

A simple web app that checks password strength in real-time and suggests improvements.

Features
  Real-time strength check (Weak / Medium / Strong)
  Password suggestions
  Show/Hide password
  Strong password generator
  Prevents password reuse (localStorage)

Tech Used
  HTML
  CSS
  JavaScript

How to Run
Open index.html in a browser.

Note
For learning purpose only. In real apps, passwords should be securely hashed.
